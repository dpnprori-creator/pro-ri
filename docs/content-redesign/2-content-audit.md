# Content Audit — PRO RI Website

> Evaluation of clarity, credibility, conversion, and storytelling for every page

---

## Scoring Rubric

| Score | Meaning |
|-------|---------|
| 1–3 | Poor — needs complete rewrite |
| 4–6 | Fair — needs significant improvement |
| 7–8 | Good — minor improvements needed |
| 9–10 | Excellent — minimal changes needed |

---

## 1. Beranda (`/`)

| Dimension | Score | Analysis |
|-----------|-------|----------|
| **Clarity** | **7/10** | "Gerakan Robotika untuk Kedaulatan Teknologi Indonesia" jelas menyampaikan misi organisasi. Namun pengunjung baru mungkin masih bertanya "Jadi PRO RI ini ngapain?" |
| **Credibility** | **6/10** | Statistik dampak bagus (38 provinsi, 6 program), tapi belum ada testimoni, mitra, atau pencapaian konkret. Organisasi baru (2026) — perlu lebih banyak social proof. |
| **Conversion** | **5/10** | CTA "Daftar Anggota Sekarang" dan "Pelajari Program Kami" ada di hero, tapi benefit konkret belum jelas. Kenapa harus daftar? Apa yang saya dapat? |
| **Storytelling** | **6/10** | Ada alur: Hero → Sekilas PRO RI → Dampak → Program → Video → Kegiatan → CTA. Namun tidak ada "mengapa sekarang" urgency. |

**Priority:** High — homepage adalah pintu utama

**Key Issues:**
- Value proposition belum cukup kuat
- Tidak ada social proof (testimoni, mitra)
- CTA tidak menyebutkan benefit
- Hero terlalu abstrak
